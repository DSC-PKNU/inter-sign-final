# Inter-Sign
Sign language exchange service


## 🌏 STACK
- FrontEnd : React Js
- BackEnd : Web RTC, Node Js
- Machine Learning : Teachable Machine


## 🌏 COMMAND
- cd client
- npx create-react-app .
- npm install react-router-dom
- npm run start (front run)


## 🌏 STRUCT
```
client 
ㄴ src
  ㄴ components - - 컴포넌트 별 js 파일
  ㄴ css - - - - -  컴포넌트 별 css 파일
  ㄴ assets
    ㄴ images
    ㄴ fonts 
  App.js - - - - -  routes 설정
  index.js
```
