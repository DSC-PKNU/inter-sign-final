# InterSign
Video chat service for international sign language exchange


## 🌏 STACK
- Front-end : HTML, CSS, Javascript, WebRTC
- Back-end : Node.js
- Hosting : Firebase
- Machine Learning : Tensorflow, Jupyter Notebook
